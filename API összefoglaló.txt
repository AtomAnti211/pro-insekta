# Frontend útvonalak összefoglaló

Ez a melléklet a PRO-INSECTA rendszer frontend útvonalait tartalmazza modulonként.

---

## 1. Publikus oldalak

- GET `/`  
  → `LandingPage` – Nyitóoldal, szolgáltatások bemutatása

- GET `/activities/:id`  
  → `ActivityDetailPage` – Publikus aktivitás részletező oldal

- GET `/login`  
  → `Login` – Bejelentkezési felület admin felhasználók számára

---

## 2. Admin – Dashboard modul

- GET `/admin/dashboard`  
  → `Dashboard` – Áttekintő admin főoldal (statisztikák, gyors linkek)

---

## 3. Admin – Notes modul

- GET `/admin/notes`  
  → `NotesAdminPage` – Jegyzetek listázása, létrehozása, szerkesztése

---

## 4. Admin – Activities modul

- GET `/admin/activities`  
  → `ActivitiesAdminPage` – Aktivitások listázása, létrehozása, szerkesztése

---

## 5. Admin – Owner modul

- GET `/admin/owner`  
  → `OwnerAdminPage` – Cégadatok (Owner) kezelése

---

## 6. Admin – Services modul

- GET `/admin/services`  
  → `ServicesAdminPage` – Szolgáltatások listázása, létrehozása, szerkesztése

---

## 7. Admin – Customers modul

- GET `/admin/customers`  
  → `CustomersAdminPage` – Ügyfelek listázása, létrehozása, szerkesztése

---

## 8. Admin – Locations modul

- GET `/admin/locations`  
  → `LocationsAdminPage` – Helyszínek listázása, térképes kezelés, szerkesztés

---

## 9. Admin – Contracts modul

- GET `/admin/contracts`  
  → `ContractsAdminPage` – Szerződések listázása, létrehozása, szerkesztése

---

## 10. Admin – Jobs modul

- GET `/admin/jobs`  
  → `JobsAdminPage` – Munkák (Job) listázása, PDF feltöltés, szerkesztés

---

## 11. Admin – Due Contracts modul

- GET `/admin/due-contracts`  
  → `DueContracts` – Esedékes szerződések listázása

---

## 🔐 Jogosultság

Az összes `/admin/...` útvonal:

- `ProtectedRoute` komponensen keresztül érhető el  
- csak bejelentkezett felhasználó fér hozzá  
- a hitelesítés a frontend `authService.ts` és a backend auth logika együttműködésével történik
