## Layout modul

A `Layout` komponens biztosítja az admin felület teljes keretrendszerét: a bal oldali navigációt, a reszponzív működést, az aktív útvonal kiemelését és a kijelentkezést. Minden admin oldal a Layout alatt jelenik meg az `Outlet` segítségével.

### Fő funkciók

- **Összecsukható / kinyitható sidebar**  
  `open` state alapján vált a *collapsed* (ikonok) és *expanded* (ikon + szöveg) mód között.

- **Aktív menüpont kiemelése**  
  A `useLocation()` figyeli az aktuális útvonalat, az `isActive()` pedig CSS‑sel jelöli a kiválasztott menüpontot.

- **Kijelentkezés**  
  A `handleLogout()` törli a JWT tokeneket a `localStorage`‑ból, majd visszairányít a `/login` oldalra.

- **Mobilbarát működés**  
  768px alatt a sidebar *slide‑in* animációval jelenik meg, alapból rejtett, a hamburger ikon nyitja meg.

- **Tartalom megjelenítése**  
  Az admin oldalak a Layout keretében töltődnek be:  
  ```tsx
  <main className="main-content">
    <Outlet />
  </main>
  ```

### Stílus és UX

- világos zöld admin‑téma (`#e8f5e9`, `#c8e6c9`, `#1b5e20`)
- ikon alapú navigáció (emoji)
- tooltip támogatás összehúzott módban
- letisztult, világos admin háttér (`#fafafa`)
- aktív elem kék kiemeléssel (`#4a6cf7`)

A Layout modul biztosítja az admin felület egységes, reszponzív és könnyen használható keretrendszerét.
